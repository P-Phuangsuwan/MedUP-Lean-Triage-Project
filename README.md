## MedUP Innovation Ideathon 2025 : Future-Proofing Healthcare

# ชื่อโครงงาน
Lean Triage Project "คัดกรองเร็ว ลดเวลา เพิ่มคุณภาพ"

# จุดประสงค์
เพื่อปรับปรุงกระบวนการคัดกรองผู้ป่วย (Triage) ในแผนกอุบัติเหตุและฉุกเฉินให้มีความรวดเร็ว ถูกต้อง และปลอดภัยตามมาตรฐานวิชาชีพ ลดระยะเวลาเฉลี่ยในการคัดกรองผู้ป่วย (Triage) โดยใช้เวลาไม่เกิน 5 นาทีต่อราย ลดความสูญเปล่า (Waste) ในกระบวนการ 
เช่น การบันทึกข้อมูลซ้ำซ้อน หรือความล่าช้าจากระบบสารสนเทศ และเพิ่มประสิทธิภาพการทำงานของบุคลากร
ลดความแออัดในห้องฉุกเฉิน และเพิ่มความพึงพอใจของผู้ป่วยและญาติ
