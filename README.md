## MedUP Innovation Ideathon 2025 : Future-Proofing Healthcare

# ชื่อโครงงาน
Lean Triage Project "คัดกรองเร็ว ลดเวลา เพิ่มคุณภาพ"

# การออกแบบหน้า UI
ประกอบด้วย 3 ผู้ใช้งาน ดังนี้
1. แพทย์ฉุกเฉิน
ฟีเจอร์หลัก : - กรอกข้อมูลปฏิบัติงานการแพทย์
           - สามารถส่งประวัติผู้ป่วยไปยังแผนกต่าง ๆ ได้
           - กรอกข้อมูลนัดผู้ป่วยและสามารถส่งไปหาผู้ป่วยโดยตรงผ่านแอปพลิเคชั่น
2. ผู้แจ้งเหตุฉุกเฉิน
ฟีเจอร์หลัก : - ในการกรอกข้อมูลจะสถานที่เกิดเหตุและเวลาเกิดจะแสดงตามผู้แจ้งเหตุและสามารถส่งข้อมูลการแจ้งฉุกเฉินผ่านแอปพลิเคชั่น
           - สามารถส่งภาพที่เกิดเหตุ เพื่อให้ตรวจสอบสถานการณ์ว่าอันตรายแค่ไหน
           - สามารถติดตามรถฉุกเฉินแบบRealtime
3. ผู้ป่วย
ฟีเจอร์หลัก : - ใบรับรองแพทย์ที่สามารถดาวน์โหลดได้ผ่านแอปพลิเคชั่น
           - สามารถดูใบนัดหมดผ่านแอปพลิเคชั่น และมีการแจ้งเตือนในวันใกล้นัดหมอ
